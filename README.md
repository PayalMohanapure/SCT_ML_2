# SCT_ML_2 - Customer Segmentation using K-Means Clustering

## Overview

This project is part of the SkillCraft Technology Machine Learning Internship.

The objective of this project is to segment retail store customers into different groups based on their annual income and spending behavior using the K-Means Clustering algorithm.

## Dataset

Customer Segmentation Dataset (Kaggle)

## Features Used

* Annual Income (k$)
* Spending Score (1-100)

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-Learn

## Machine Learning Model

* K-Means Clustering

## Project Workflow

1. Load the dataset
2. Perform data exploration
3. Select relevant customer features
4. Apply the Elbow Method to determine the optimal number of clusters
5. Train the K-Means clustering model
6. Assign customers to clusters
7. Visualize customer segments
8. Export clustered dataset

## Visualizations

* Elbow Method Graph
* Customer Cluster Visualization
